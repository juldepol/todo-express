This is the simple ToDo list app build with Express, SQLite and React.

![App](https://github.com/juldepol/todo-express/blob/master/sreenshots/app.png)

Run backend
-----------
#### Set up
    cd server 
    npm install
#### Run
    npm start
#### Tests              
    npm test

Run frontend
------------
#### Set up
    cd frontend 
    npm install
#### Run
    npm start
Visit http://localhost:3000/
#### Tests
    npm test
